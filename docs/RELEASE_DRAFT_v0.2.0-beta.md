# PC Power Free v0.2.0-beta

Beta release for the first public test cycle.

## Included

- Home Assistant custom integration
- Automatic LAN discovery with zeroconf
- Pairing with a temporary code
- Wake-on-LAN power on
- Local shutdown and restart through the Windows agent
- Windows installer and standalone binaries
- English and Spanish setup UI

## Assets to attach

- `pcpowerfree-windows-x64-setup.exe`
- `PCPowerAgent.exe`
- `PCPowerSetup.exe`

## Known status

Feature-complete, but still pending real-world testing with:

- Home Assistant discovery
- Pairing flow
- Wake-on-LAN on actual hardware
- Alexa through Home Assistant `emulated_hue`

## Notes

Use this as a prerelease or beta until the first successful end-to-end validation on a real setup.
